# DeLLa
> DLL injector
## Features
- Uses two methods of injection: Kernell LoadLibrary and Manual mapping
- Logs process of injection
- Simple UI
## Dependancies
 - Lunar
## Installation
 dotnet add PROJECT package Lunar --version 1.4.10
